--- title: How are errors corrected? layout: page ---

Lists often contain errors, omissions or inaccuracies. This is particularly true of lists which are reality describing and thus will often not have changed as fast as reality. When this happens, we need a way to make changes to that list.

Direct access

When the process for correcting an error is direct access custodians can describe this using language such as:

Users of Open Street Map may correct an error in the map directly using the editing features of the platform itself.

Suggested edits

When the process for correcting an error is suggested edits custodians can describe this using language such as:

Anyone may suggest a specific alteration to the England Local Authorities register. The source data files for the register are published on the web on a platform which allows anyone to suggest a change.

Error Reporting

When the process for correcting an error is error reporting custodians can describe this using language such as:

Anyone may report an error in the List of historic buildings maintained by Historic England. Reports will be investigated and the list updated if an error is found.
