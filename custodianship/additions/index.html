How are entries added?

What is the process for adding an entry to the list? Which people, tools or institutions will be involved, will there be an approval step or can I add one directly?

Direct access

When the process for adding an entry is direct access custodians can describe this using language such as:

Users of Open Street Map may add an entry to the map directly using the editing features of the platform itself.

Suggested edits

When the process for adding an entry is suggested edits custodians can describe this using language such as:

When you have identified a list that should be included in org-id.guide, you can post a "PROPOSAL’ issue, or submit a pull-request with a fully-researched list.

This example taken directly from the org-id.guide contributor handbook .

Custodial discretion

When the process for adding an entry is custodial discretion custodians can describe this using language such as:

When the UK Government decides to recognise a new country, the country will be added to the register.
